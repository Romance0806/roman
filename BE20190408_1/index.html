  2. 當對方同意對話後，點擊配對挑戰，就會顯示等級區間與經驗值分配

    3. ＊經驗值分配需獲得雙方同意

  3. 基本設定

    4. - 副本入場次數為3次，每天00:00自動重置

    - 約會地下城可隨機入場

    - 入場後右下方會顯示剩餘時間，下方顯示目前約會地區的資訊

    - 進行途中任一方結束遊戲，另一方會自動從所在的地圖離開

    - 約會副本可獲得大量經驗

  4. 約會地圖

  1. 透過奈拉城 NPC "帕樂非"報名參賽與其他功能
  2. 和NPC對話可以查看公會聯賽相關資訊
  3. 報名時間為每周四00:00~21:50
  4. 滿足以下狀況時決定勝負並結束比賽

    5. -比賽時間結束: 高分隊伍獲勝 (同分時隨機獲勝)

    -達到比賽目標分數: 達成隊伍獲勝

    -所有成員離開隊伍: 留下的公會獲勝

    -獲勝後約5秒會移動到奈拉城

  5. 聯賽開始前3分鐘會出現進場訊息，也可以透過NPC進場
  6. 聯賽正式開始可以透過上方查看雙方分數及剩餘時間

    7. - 左/右 分數

    - 上方目標分數

    - 下方比賽進行時間會獲勝

  1. 進入遇難船

    2. 遇難船難度分為初級、中級、高級、困難，建議等級為85、90、95、100級可透過喲格梅爾入場。

  2. 組隊狀態

    3. a.進入副本需要組隊

    b.副本內被踢出或退隊，一段時間後會強制離開副本

    c.組隊狀態下可單獨入場

  3. 入場次數限制

    4. 副本入場次數為2次，每天00:00自動重置

  4. BOSS設定

    5. a.BOSS房間有巨大的毒井，若在井內會受到持續的毒效果。

    b.BOSS被召喚時會一同召喚4個守護石碑，在守護石碑存在期間BOSS會呈現無敵狀態。

    c.最後1個守護石碑被破壞時，會有1分鐘的冷卻時間。在這1分鐘可對BOSS進行攻擊。

    d.若靠近守護石碑時，會召喚4隻毒氣球。(冷卻時間1分鐘)

    e.依照BOSS的HP會召喚不同的怪物，並對全區施放減益效果。

  1. 進入貪念的寺院

    2. 貪念的寺院難度分為初級、中級建議等級為100以上。可透過喲格梅爾入場。

  2. 組隊狀態

    3. a.進入副本需要組隊

    b.副本內被踢出或退隊，一段時間後會強制離開副本

    c.組隊狀態下可單獨入場

  3. 入場次數限制

    4. 副本入場次數為2次，每天00:00自動重置

  4. BOSS設定

    5. a.BOSS用的詛咒可利用封印石解除，要善加利用。

    b.BOSS體力剩 70%時，副本內所有玩家會受到減益效果。

    c.施放毒氣效果時，周圍會出現4個封印石。

    d.啟動所有封印石，就能解除減益效果。

    e.BOSS體力剩 60%就會形成防護罩，並成為無敵狀態。

    f.形成防護罩的同時會召喚怪物，消滅所有怪物可解除防護罩。

    g.BOSS體力剩 50%，所有地須會變成死亡地區。

    h.變成死亡地區時會出現3個封印石，啟動所有封印石後中央會出現安全地區。

    i.若BOSS召喚怪物時，由於外圍變成死亡地區，因此需要將怪物引到安全地區擊殺。
